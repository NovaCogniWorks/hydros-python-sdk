# Document Archive Manifest

本目录是 `agent_workspace` 的统一文档归档区。

归档原则：

- 采用复制归档，不移动原始文件
- 保持原工作路径可继续使用
- 按文档职能分类归档，便于检索与交付

分类说明：

- `00-workspace-overview`：工作区总览与索引
- `01-role-guides`：角色说明与工作目录说明
- `02-architect-inputs`：架构分析输入文档
- `03-architect-outputs`：架构设计输出文档
- `04-validation-artifacts-docs`：脚手架验证样本相关文档
- `05-execution-docs`：执行侧正式文档与脚本归档
- `06-delivery-package-docs`：对外交付包文档
- `07-sample-configs-and-commands`：样例配置与联调命令
- `08-shared-context`：共享模板与上下文规范
- `09-logs-and-governance`：日志与治理说明
- `10-engineering-guides`：工程智能体说明文档

本轮已同步的重点内容：

- 最新版 `README.md` 与 `INDEX.md`
